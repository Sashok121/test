Требования:
1. Выполнить верстку макета в .html-файл, используя технологии  css+html+javascript и согласно стандаров в документе https://drive.google.com/file/d/0B8_ubqXnxZH8Z3BxQ3RSWWpRUW8/.
2. Кодировка файлов должна быть utf-8.
3. Верстка сайта должна быть кроссбраузерной и обеспечивать одинаковое корректное  отображение сайта в браузерах Internet Explorer v.8+, Mozilla Firefox v3.5+, Opera v9.0+, Safari v3.0+, Chrome v5.0+.. 
Для остальных, более старых версий браузеров, обязательно должна появляться информация на сайте о том, что необходимо обновить браузер для корректного отображения сайта.
4. Минимальная ширина макета без горизонтального скроллинга 1000 пикселей.
5. Результат должен содержаться в папке site1 cо след. структурой:
	/index.html - основной файл с версткой.
	/favicon.ico - иконка страницы для барузера.
	/images/ - папка, содержащая все изображения, использованные в верстке.
	/js/ - папка, содержащая файлы javascript.
	/css/ - папка, содержащая файл стилей.

Примечания:
В верхнем горизонтальном меню показать выпадающее меню. Оформление и стили на ваш вкус.
Большая картинка под горизонтальным меню это слайдер - должен изменять изображение и текст на голубом фоне.
Кнопка "Отправить заявку" - всплывающее окно формы  обратной связи с полями:
Имя
Емейл
Телефон
Сообщение
Оформление всплывающего окна на ваш вкус.
В этой части задания отправку формы выполнять не нужно.
